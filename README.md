# Obsidian Notes – TechVault Backup

Welcome! 🎉  
This repository contains interconnected notes designed for use with [Obsidian](https://obsidian.md/).  
To get the best reading and navigation experience, please follow the instructions below.

---

## 📥 How to Use

1. **Install Obsidian**  
   Download and install Obsidian from [obsidian.md](https://obsidian.md/download).

2. **Clone or Download this Repository**  
   - Clone: `git clone https://github.com/Byte2Go/techvault-backup.git`  
   - Or download the ZIP and extract it locally.

3. **Open in Obsidian**  
   - Launch Obsidian.  
   - Select **Open Folder as Vault**.  
   - Choose the folder containing these notes.

---

## 🔗 Why Obsidian?

- All notes are **interconnected** using Obsidian’s graph view and backlinks.  
- The structure is optimized for **Obsidian compatibility** (markdown format, internal links).  
- You’ll get the full benefit of **bidirectional linking**, **graph visualization**, and **search features**.

---

## ⚡ Tips

- Use the **Graph View** to explore relationships between notes.  
- Try the **Backlinks panel** to see contextual references.  
- Customize with community plugins for even richer navigation.
